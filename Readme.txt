directory summary:

cfg.mk
------
some directory varibalbes

component
---------
third party usage component, such as qt, libgphoto2, 

patch
-----
all modified files that we changed

rootfs
------
rootfs base and tools. such busybox, gdb, some utilities

source
------
component original source archive and mirror website where we download these files

sys-root
-------- 
base system rootfs that was generated by petalinux-2016.2 . system lib and some components (example libc.so,libopencv.so)  could be copied from there.

petalinux
---------
petalinux project and some useful already built libs

tools
-----
cross compile tools and some useful tool script

uboot
-----
boot compile dir

kernel
------
kernel compile dir

app
---
some sample test app dir
